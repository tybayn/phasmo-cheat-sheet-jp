音声コマンド

音声コマンドはチートシートを操作することが可能。「有効」をクリックすると音声コマンドがツールにリセットした際でも無効として設定するまで入力される。デフォルトマイクから入力されるので、環境上の設定をご確認ください。本ツールが最小化されても音声コマンドが、同じマイクをゲーム内で使用しても、有効となり入力される。
証拠:
  • "難易度 <難易度>" -> 難易度を設定する
  • "証拠数<数字>" -> カスタム: 証拠数
  • "ハント時間 <数字>" -> カスタム: ハント時間を設定する
  • "証拠 <証拠>" -> 見つけた証拠として記録する
  • "証拠 ではない <証拠>" -> 証拠ではないとして記録する
  • "証拠 クリア <証拠>" -> 証拠の状況をリセットする
  • "猿の手 <証拠>" -> 「猿の手」の「知識が欲しい」に対する証拠ではないとされた証拠として記録する
速度:
  • "速度 <速度>" -> 特定の速度を記録する
  • "速度 クリア <速度>" -> 特定の速度の状況をリセットする
  • "速度 視認加速 がある" -> 通常視認加速であるとして記録する
  • "速度 視認加速 がない" -> 通常視認加速ではないとして記録する
  • "速度 クリア 視認" -> 視認加速フィルターをリセットする
ハント 正気度:
  • "正気度 <正気度>" -> ハントされた正気度を記録する
  • "正気度 クリア <正気度>" -> 特定のハントされた正気度の状況をリセットする
ゴースト:
  • "ゴースト ではない <ゴースト>" -> 特定されたゴーストのカードを塗りつぶす
  • "ゴースト クリア <ゴースト>" -> ゴーストのカードの状況をリセットする(非表示カードも含め)
  • "ゴースト 選択 <ゴースト>" -> 特定のゴーストカードをハイライトする
  • "ゴースト 削除 <ゴースト>" -> 特定のゴーストカードを非表示にする
  • "ゴースト 見せる|概要 <ゴースト>" -> 特定のゴーストカードを見えるようにする
タイマー:
  • "タイマー スタート" -> スマッジ(浄化香)タイマーを開始する
  • "タイマー ストップ" -> スマッジ(浄化香)タイマーを停止する
  • "クールダウン スタート" -> クールダウンタイマーを開始する
  • "クールダウン ストップ" -> クールダウンタイマーを停止する
  • "ハント スタート" -> ハントタイマーを開始する
  • "ハント ストップ" -> ハントタイマーを停止する
その他:
  • "表示 フィルター" or "表示 ツール" -> メニューをフィルター/ツールに切り替える
  • "ゴースト 速度 <パーセント>" -> ゴーストの速度調整を50%、75%、100%、125%、または150%に設定する
  • "選択 マップ<マップ名>" -> 特定のマップを選択
  • "表示 マップ <マップ名>" -> マップのタブを開き、マップを表示する
  • "クローズ マップ" or "非表示 マップ" -> マップを非表示する
  • "リセット ジャーナル" or "リセット チートシート" -> チートシートをリセットする
  • "ストップ 入力" -> 音声コマンドを停止する

データ プライバシー

音声コマンドは記載されているWeb Speech APIを通して処理されている。本機能を使用の際、マイクのアクセスは必要だが、音声は録音および保存はされていません。
